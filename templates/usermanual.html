How To Use ?

This application is designed to melanoma skin cancer image with explainable heat maps. In this user manual, will explain how to use the web application in step by step .


STEP 01- Edit & Select the skin image

To use the application, the first step is to upload the skin image.Before the uploading image image if you need edit that , use image editer by checking check box. After the editing , you can save that image and upload it again for prediction ( File formats: .jpeg .jpg, .png )

STEP 02- Choose the Model

After the successfully uploading image, choose AI model which you are going to use to predction.(Deafult use CNN model)

CNN Model : Convolutional Neural Networks (Xception model)
VIT Model : Vision Transfomer (Salient Mask-Guided Vision Transformer)

STEP 03- Generate the prediction with Heat maps

Click prediction button and wait for the model results. As the model result , you can see predicted skin cancer and generated heatmaps with prediction accuracy. (To go back click 'Clear' button and try new skin image)

STEP 04- Download the results report

Finally,Can generate and download the medical report (PDF) with dermatologist feedbacks.
